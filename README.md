# PHP Labs — единый каркас

Проект объединяет лабораторные работы и контрольную в едином веб-приложении с переключаемыми стилями.

## Запуск в Docker
```
docker compose up -d
```
Откройте: http://localhost:8008/

Образы собираются из `docker/Dockerfile`; корень сайта — `public/`. Проект примонтирован как volume, поэтому изменения видны сразу.

## Структура
- `public/index.php` — главная страница со ссылками на лабораторные.
- `includes/header.php`, `footer.php`, `nav.php` — общий каркас, верхнее меню и переключатель темы (cookie `theme`).
- `public/assets/css/old.css`, `modern.css` — два варианта дизайна; переключатель в шапке.
- `public/assets/js/theme.js` — логика смены дизайна.
- `public/labs/lr1...lr4`, `public/labs/kr` — упражнения и контрольные задания.

## Навигация
Верхнее меню ведёт на Главную, ЛР1–ЛР4 и КР, при наведении раскрываются упражнения. Персонализация автора вынесена в шапку.
